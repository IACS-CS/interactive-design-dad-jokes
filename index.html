The Bowline (Fixed Loop) Knot

     Fixed Loop (Bowline)
     
          Working End
                 |
                 v
         ________|________
        /                 \
       |      Main Line    |
       |                   |
       |    /------Loop    |
       |   /      (fixed)  |
       |  /               /
       | /_______________/
       |
       |---- Used for tying onto objects
       |---- Creates a loop that won't slip
       |---- Essential for climbing & sailing

How the Bowline Works:

The Bowline creates a fixed loop at the end of a rope. Unlike a slip knot, the loop will not tighten or loosen - it stays the same size. This makes it perfect for:

  • Tying a rope to a ring or post
  • Creating a loop for climbing
  • Securing sails on boats
  • Any situation where you need a dependable, fixed loop

How to Tie a Bowline:

  1. Make a small loop in the main part of the rope (not the end)
  2. Thread the working end (the loose end) up through the loop
  3. Wrap it around the main line behind and back down
  4. Push it back through the loop from the same side it came up
  5. Pull tight on the working end and main line
  6. The loop at the top is now your fixed, non-slip loop

Tip: A common saying is "The rabbit comes out of the hole, around the tree, and back down the hole" - this describes the motion of the rope!

The Buntline (Sliding Loop) Knot

     Sliding Loop (Buntline)
     
          Working End
                 |
                 v
         ________|________
        /                 \
       |      Main Line    |
       |                   |
       |    /---Loop----   |
       |   /   (slides)  \ |
       |  /               \|
       | /____Loop Tightens_\
       |
       |---- Loop slides along the rope
       |---- Used for bundling items
       |---- Creates adjustable loops

How the Buntline Works:

The Buntline creates a sliding loop that can be tightened or loosened by pulling the working end. Unlike the fixed Bowline, this loop can move along the rope and change size. This makes it perfect for:

  • Bundling items together (like sailbags or gear)
  • Creating adjustable loops that can get tighter
  • Securing loose items that need different tension
  • Quick, flexible knots that are easy to untie

How to Tie a Buntline:

  1. Form a loop with the working end of the rope
  2. Bring the working end up through the loop from underneath
  3. Wrap it around the main line (going back behind itself)
  4. Tuck the working end back into the loop
  5. Pull the main line to tighten - the loop will slide and grip
  6. The tighter you pull, the more the loop grips without slipping

Tip: The Buntline is great for "cinching" loads because the more you pull, the tighter it gets - making it very secure even under stress!
